## [1.2.0] - 2026-01-02

### Added
- **Aggressive Governor Tunables**: Optimasi parameter `schedutil` dan `interactive` untuk ramping frekuensi instan.
- **Surface Flinger Boost**: Memaksa GPU composition untuk UI rendering guna mengurangi micro-stutter.
- **Process Prioritization**: Implementasi prioritas tugas level kernel untuk aplikasi gaming di foreground.
- **I/O Throughput Enhancement**: Peningkatan kecepatan akses database untuk peluncuran aplikasi yang lebih cepat.

### Changed
- **Thermal Engine Management**: Pembaruan logika bypass thermal untuk mendukung lebih banyak layanan vendor MediaTek dan Snapdragon.
- **Hard-Lock Logic**: Peningkatan skrip deteksi frekuensi maksimum pada chipset Dimensity dan Snapdragon terbaru.
- **Module Identity**: Pembaruan `module.prop` dan `customize.sh` ke versi 1.2.0.

### Fixed
- Memperbaiki isu di mana beberapa Little cores turun ke posisi idle pada perangkat MediaTek tertentu.
- Menyelesaikan konflik dengan `logd` yang menyebabkan penggunaan CPU berlebih saat sesi performa tinggi.

### Removed
